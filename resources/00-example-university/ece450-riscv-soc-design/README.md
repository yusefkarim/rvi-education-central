# RISC-V SoC Design (ECE450)

Graduate-level companion course to CPSC 313, added to show how a single
institution hosts more than one course and to exercise the metadata features the
undergraduate course leaves untouched. It is bilingual (English and German), sets
its own `license: MIT` at the course level, and its figures demonstrate filename
language suffixes, sidecar `topic` and `lang` overrides, and an `attribution`
line. As with the rest of Example University, the diagrams are original
placeholders, not real lecture material.

Three short lecture units cover pipeline hazards and forwarding in
`lectures/01-pipeline-hazards`, the vector extension in
`lectures/02-vector-extension`, and privileged-mode trap handling in
`lectures/03-privileged-modes`. A toolchain screencast is referenced from
`course.yml` as an external asset rather than committed.
