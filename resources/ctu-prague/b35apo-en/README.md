# Computer Architectures (B35APO)

English-language variant of the undergraduate computer architecture course
at CTU Prague, covering number representation, digital logic, the CPU
datapath, memory, pipelining, I/O, and virtualization. Companion simulator:
[QtRVSim](https://github.com/cvut/qtrvsim).

Only `lectures/01-intro` is migrated so far; see the resource-level README
for the status of this migration.
