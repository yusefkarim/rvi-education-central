# Accepted licenses

This repository accepts a fixed allowlist of open licenses, enforced by the
`license` enum in every schema under `schema/`. Each entry below links to the
canonical, authoritative legal text; full license text is deliberately not
vendored into this repository so this directory never goes stale relative to
the license steward's own copy.

| Identifier | Name | Typical use |
|---|---|---|
| `CC-BY-4.0` | Creative Commons Attribution 4.0 | Slides, figures, notes |
| `CC-BY-SA-4.0` | Creative Commons Attribution-ShareAlike 4.0 | Slides, figures, notes (repository default) |
| `MIT` | MIT License | Lab and example source code |
| `Apache-2.0` | Apache License 2.0 | Lab and example source code |
| `BSD-3-Clause` | BSD 3-Clause License | Lab and example source code |
| `none` | No license granted | Hosted for course use only, excluded from `common/` pooling |

Canonical texts:

- CC-BY-4.0: https://creativecommons.org/licenses/by/4.0/legalcode
- CC-BY-SA-4.0: https://creativecommons.org/licenses/by-sa/4.0/legalcode
- MIT: https://opensource.org/license/mit
- Apache-2.0: https://www.apache.org/licenses/LICENSE-2.0
- BSD-3-Clause: https://opensource.org/license/bsd-3-clause

See `PLAN.md` section 6 for the license cascade and the repository-wide
default, and `CONTRIBUTING.md` for how the default applies to your pull
request.
