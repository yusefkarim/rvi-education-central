# RVI Education Central

A central home for RISC-V lecture and lab material, shared and reused across
universities, institutions, and companies. Material lives under `resources/`,
one directory per contributing institution; shareable figures are pooled by
topic under `common/`. See `CONTRIBUTING.md` to add your own material.

This file, including the institution listing below, is generated by
`_tools/gen_readme.py` from the descriptors under `resources/` - edit those,
NOT this file.

## Core ideas

- One shared home for RISC-V teaching material, so universities and companies can reuse each other's lectures and labs instead of each building the same thing from scratch.
- Every contributor keeps their own space: one folder per institution, and material can be added in whatever format you already teach with.
- Figures and diagrams are gathered by topic, so anyone can pull a good illustration (a CPU pipeline, a memory layout) straight into their own slides.
- Material is openly licensed by default, which is what makes that reuse legal; you can opt out and just host without sharing.
- Getting listed takes only a few minutes, and going further (documentation, licensing, pooled figures) earns a higher badge on a public scorecard, but is never required.
- The catalog you see below is generated automatically from what contributors add, so it stays current on its own.

## Shared asset pool

Figures and diagrams contributed under an open license are pooled by topic under [`common/`](common/); browse a topic's `README.md` for a gallery, for example [`common/cpu`](common/cpu).

See [`STANDARDS.md`](STANDARDS.md) for the per-institution compliance scorecard.
