# RVI Education Central

A central home for RISC-V lecture and lab material, shared and reused across
universities, institutions, and companies. Material lives under `resources/`,
one directory per contributing institution; shareable figures are pooled by
topic under `common/`. See `CONTRIBUTING.md` to add your own material.

This file, including the institution listing below, is generated by
`_tools/gen_readme.py` from the descriptors under `resources/` — edit those,
not this file.
