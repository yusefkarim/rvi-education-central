# Compliance scorecard

Every institution directory earns a tier automatically from what it has
adopted; nothing here is assigned by hand. Generated by
`tools/scorecard.py` — see `PLAN.md` section 8 for what each tier means and
how to move up.

| Tier | Requirement |
|---|---|
| ⬜ Listed | Directory name is registered in `institutions.yml`. |
| 🥉 Structured | `resource.yml` and every `course.yml` are present and valid. |
| 🥈 Documented | The institution and every course have a substantive `README.md`. |
| 🥇 Pooled | At least one asset is admitted into `common/`. |
